# プロジェクト初期構築時コマンド
```
$ create-react-app .
$ npm i --save material-ui@next
$ npm i --save material-ui-icons
$ npm i --save react-transition-group@1.x
```

# ビルド
```
$ npm run build
```

# アプリケーション起動
```
$ npm run start
```
